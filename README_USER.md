# DMarket Trading Bot - Інструкція для користувачів

## Що це таке?

**DMarket Trading Bot** - це настільний додаток для автоматичного управління вашими оферами та таргетами на торговому майданчику DMarket.

### Основні можливості:

✅ **Автоматичне управління таргетами (buy orders)**
- Автоматичне оновлення цін таргетів
- Встановлення максимальної ціни для кожного таргета
- Активація/деактивація таргетів
- Відстеження float значень предметів

✅ **Автоматичне управління оферами (sell offers)**
- Автоматичне оновлення цін оферів
- Встановлення мінімальної ціни для кожного офера
- Видалення оферів

✅ **Аналітика та статистика**
- Графіки продажів та покупок за період (7, 30, 60 днів)
- Розрахунок прибутковості та маржі прибутку
- Топ найприбутковіші предмети
- Список останніх транзакцій
- Детальна статистика по днях

✅ **Сповіщення**
- Сповіщення про успішні продажі та покупки
- Попередження про низький баланс
- Сповіщення про помилки API
- Інтеграція з Telegram для отримання сповіщень
- Внутрішні сповіщення в додатку

✅ **Моніторинг транзакцій**
- Автоматична перевірка нових транзакцій кожні 10 хвилин
- Відстеження продажів, покупок та закритих таргетів
- Автоматичне додавання транзакцій до аналітики

✅ **Система логування**
- Всі операції записуються в логи
- Можливість фільтрації та пошуку в логах

✅ **Безпечне зберігання ключів**
- Ваші API ключі зберігаються безпечно
- Кожен користувач вводить свої власні ключі

## Як встановити та запустити

### Крок 1: Завантажте та встановіть

#### 📱 Для Windows:

1. Завантажте файл `DMarket Bot Setup.exe`
2. Запустіть інсталятор (може знадобитися дозвіл адміністратора)
3. Слідуйте інструкціям встановлення:
   - Ви можете вибрати папку встановлення
   - Можна створити ярлик на робочому столі
   - Можна додати в меню "Пуск"
4. Після встановлення додаток з'явиться в меню "Пуск" та на робочому столі (якщо вибрали цю опцію)
5. Запустіть додаток з меню "Пуск" або двічі клацніть на ярлик

**Примітка:** При першому запуску Windows може попросити підтвердження безпеки - натисніть "Так" або "Запустити в будь-якому випадку".

#### 🍎 Для macOS:

1. Завантажте файл `DMarket Bot-0.7.1.dmg`
2. Відкрийте завантажений `.dmg` файл (двічі клацніть на нього)
3. У відкритому вікні перетягніть **DMarket Bot** в папку **Applications** (Програми)
4. Після копіювання можете закрити вікно DMG та видалити файл `.dmg` (він більше не потрібен)
5. Відкрийте папку **Applications** (Програми) та запустіть **DMarket Bot**

**Примітка:** При першому запуску macOS може показати попередження про безпеку:
- Перейдіть в **Системні налаштування** → **Безпека та конфіденційність**
- Натисніть **"Відкрити в будь-якому випадку"** біля повідомлення про DMarket Bot
- Або клацніть правою кнопкою на додаток та виберіть **"Відкрити"**, потім підтвердіть

**Альтернативний спосіб (без встановлення):**
- Можна запускати додаток безпосередньо з `.app` файлу з папки `dist-electron/mac/`
- Але рекомендується встановити в Applications для зручності

### Крок 2: Отримайте API ключі від DMarket

1. Увійдіть на [DMarket](https://dmarket.com)
2. Перейдіть в налаштування профілю
3. Знайдіть розділ "API Keys" або "API ключі"
4. Створіть нові ключі або скопіюйте існуючі:
   - **Public Key** (Публічний ключ)
   - **Secret Key** (Приватний ключ)

⚠️ **Важливо:** Зберігайте ключі в безпеці! Не діліться ними з іншими.

### Крок 3: Налаштуйте додаток

1. Запустіть **DMarket Bot**
2. Відкриється екран авторизації
3. Введіть ваші **Public Key** та **Secret Key**
4. Натисніть "Увійти"

Після успішної авторизації ви побачите головне меню.

## Як користуватись

### Головне меню

У лівій частині екрану знаходиться меню з табами:

- **📋 Таргети** - управління buy orders
- **💰 Офери** - управління sell offers
- **📊 Аналітика** - статистика продажів, покупок та прибутковості
- **🔔 Сповіщення** - перегляд та налаштування сповіщень
- **📋 Логи** - перегляд всіх операцій
- **⚙️ Налаштування** - зміна API ключів

### Робота з таргетами (Buy Orders)

1. Перейдіть на вкладку **"📋 Таргети"**
2. Ви побачите список всіх ваших активних таргетів
3. Для кожного таргета можна:
   - **Встановити максимальну ціну** - введіть значення в поле "Макс. ціна" та натисніть ✓
   - **Змінити кількість** - просто введіть нове значення в поле кількості
   - **Активувати/Деактивувати** - натисніть кнопку ▶ або ⏸
   - **Видалити** - натисніть кнопку 🗑
4. **Автоматичне оновлення цін:**
   - Натисніть кнопку **"▶ Запустити парсинг таргетів"** в лівому меню
   - Бот автоматично буде перевіряти ціни інших таргетів раз на хвилину
   - Якщо знайде більш високу ціну (в межах вашої максимальної), встановить вашу ціну на 1 цент вище

### Робота з оферами (Sell Offers)

1. Перейдіть на вкладку **"💰 Офери"**
2. Ви побачите список всіх ваших активних оферів
3. Для кожного офера можна:
   - **Встановити мінімальну ціну** - введіть значення в поле "Мін. ціна" та натисніть ✓
   - **Видалити** - натисніть кнопку 🗑
4. **Створити нові офери:**
   - Натисніть кнопку **"Створити офери"**
   - Виберіть предмети з вашого інвентарю
   - Встановіть ціни
   - Натисніть "Створити"
5. **Автоматичне оновлення цін:**
   - Натисніть кнопку **"▶ Запустити парсинг оферів"** в лівому меню
   - Бот автоматично буде перевіряти ціни інших оферів раз на хвилину
   - Якщо знайде більш низьку ціну (вище вашої мінімальної), встановить вашу ціну на 1 цент нижче

### Аналітика та статистика

1. Перейдіть на вкладку **"📊 Аналітика"**
2. Ви побачите:
   - **Загальний прибуток** - сумарний прибуток за обраний період
   - **Загальні продажі** - сума всіх продажів
   - **Загальні покупки** - сума всіх покупок
   - **Топ найприбутковіші предмети** - список предметів з найбільшим прибутком
   - **Графік продажів та покупок** - візуалізація транзакцій по днях
   - **Список останніх транзакцій** - останні 15 транзакцій
3. Можна вибрати період для аналізу:
   - 7 днів
   - 30 днів
   - 60 днів
4. Натисніть **"Завантажити транзакції"** для завантаження історії з DMarket API

### Сповіщення

1. Перейдіть на вкладку **"🔔 Сповіщення"**
2. У вкладці **"Сповіщення"** ви побачите всі сповіщення:
   - Про успішні продажі та покупки
   - Про низький баланс
   - Про помилки API
3. У вкладці **"Налаштування сповіщень"** можна:
   - **Налаштувати типи сповіщень:**
     - Сповіщення про продажі
     - Сповіщення про покупки
     - Попередження про низький баланс
     - Сповіщення про помилки API
     - Ринкові можливості
   - **Встановити поріг низького балансу** - мінімальна сума, при якій спрацює попередження
   - **Налаштувати Telegram:**
     - Увімкнути/вимкнути інтеграцію
     - Ввести Bot Token (отримайте від @BotFather в Telegram)
     - Ввести Chat ID (ваш ID або ID каналу)
   - **Протестувати сповіщення** - кнопки для тестування без реальних транзакцій

**Як отримати Telegram Bot Token:**
1. Відкрийте Telegram та знайдіть бота @BotFather
2. Надішліть команду `/newbot`
3. Слідуйте інструкціям для створення бота
4. Скопіюйте отриманий токен та вставте в додаток

**Як отримати Chat ID:**
1. Знайдіть бота @userinfobot в Telegram
2. Надішліть команду `/start`
3. Бот надішле вам ваш Chat ID
4. Скопіюйте ID та вставте в додаток

### Перегляд логів

1. Перейдіть на вкладку **"📋 Логи"**
2. Ви побачите всі операції, які виконував бот:
   - Створення/оновлення/видалення таргетів та оферів
   - Автоматичні оновлення цін
   - Помилки та попередження
3. Можна фільтрувати логи за:
   - Типом (Інформація, Успіх, Помилка, Попередження)
   - Категорією (Таргет, Офер, Парсинг, Система)
   - Пошуком за текстом

### Налаштування

1. Перейдіть на вкладку **"⚙️ Налаштування"**
2. Тут можна:
   - **Змінити API ключі** - введіть нові ключі та натисніть "Зберегти ключі"
   - **Вийти з системи** - видалить збережені ключі (потрібно буде ввести їх знову)

## Як працює автоматичне оновлення цін

### Для таргетів (Buy Orders):

1. Бот перевіряє всі інші таргети на той самий предмет
2. Знаходить найвищу ціну серед них (але не більше вашої максимальної)
3. Встановлює вашу ціну на **1 цент вище** знайденої
4. Перевірка відбувається **раз на хвилину**

**Приклад:**
- Ваш таргет: AK-47 | Redline, макс. ціна: $50
- Інші таргети: $45, $46, $47
- Бот встановить вашу ціну: **$47.01**

### Для оферів (Sell Offers):

1. Бот перевіряє всі інші офери на той самий предмет
2. Знаходить найнижчу ціну серед них (але не нижче вашої мінімальної)
3. Встановлює вашу ціну на **1 цент нижче** знайденої
4. Перевірка відбувається **раз на хвилину**

**Приклад:**
- Ваш офер: AK-47 | Redline, мін. ціна: $40
- Інші офери: $50, $49, $48
- Бот встановить вашу ціну: **$47.99**

## Моніторинг транзакцій

Додаток автоматично відстежує ваші транзакції на DMarket:

- **Перевірка нових транзакцій** відбувається **кожні 10 хвилин**
- Автоматично виявляються:
  - Успішні продажі
  - Покупки
  - Закриті таргети
- Всі транзакції автоматично додаються до аналітики
- Ви отримуєте сповіщення про нові транзакції (якщо увімкнено)

## Поради та рекомендації

### Безпека

- ✅ Ніколи не діліться своїми API ключами
- ✅ Регулярно перевіряйте логи на наявність помилок
- ✅ Якщо підозрюєте компрометацію ключів - змініть їх на DMarket

### Оптимальне використання

- ✅ Встановлюйте розумні максимальні/мінімальні ціни
- ✅ Перевіряйте логи регулярно
- ✅ Не запускайте парсинг для великої кількості предметів одночасно (може бути занадто багато запитів)

### Розв'язання проблем

**Проблема:** Бот не оновлює ціни
- Перевірте, чи запущений парсинг (кнопка має бути червоною)
- Перевірте логи на наявність помилок
- Переконайтеся, що встановлені мінімальні/максимальні ціни

**Проблема:** Помилки авторизації
- Перевірте правильність API ключів
- Переконайтеся, що ключі активні на DMarket
- Спробуйте вийти та увійти знову

**Проблема:** Додаток не запускається

**Windows:**
- Перевірте, чи встановлено всі необхідні компоненти
- Спробуйте перевстановити додаток
- Перевірте, чи немає конфліктів з антивірусом
- Спробуйте запустити від імені адміністратора

**macOS:**
- Якщо додаток не відкривається з попередженням про безпеку:
  1. Клацніть правою кнопкою (або Control + клац) на додаток в Applications
  2. Виберіть **"Відкрити"**
  3. Підтвердіть відкриття в діалоговому вікні
- Переконайтеся, що ви встановили додаток в папку Applications (не запускаєте з DMG)
- Перевірте налаштування безпеки в **Системні налаштування** → **Безпека та конфіденційність**

# Інструкції зі збірки та публікації

## Як запустити збірку в GitHub Actions

### Спосіб 1: Через GitHub UI (найпростіший)

1. Відкрийте репозиторій на GitHub: `https://github.com/Yarik-osg/Dmarket_js_bot`
2. Перейдіть на вкладку **"Actions"** (вгорі сторінки)
3. У лівому меню виберіть workflow **"Build Desktop App"**
4. Натисніть кнопку **"Run workflow"** (справа вгорі)
5. У випадаючому меню:
   - **Platform**: виберіть `mac` (для macOS) або `all` (для всіх платформ)
   - Натисніть **"Run workflow"**
6. Дочекайтеся завершення збірки (зазвичай 5-10 хвилин)
7. Після завершення:
   - Натисніть на завершений workflow run
   - Прокрутіть вниз до секції **"Artifacts"**
   - Завантажте файл `macos-dmg` (для macOS) або `windows-installer` (для Windows)

### Спосіб 2: Через створення тегу (автоматично)

1. Переконайтеся, що всі зміни закомічені та запушені:
```bash
git add .
git commit -m "Update to version X.X.X"
git push
```

2. Створіть новий тег (замініть X.X.X на версію, наприклад 0.7.2):
```bash
git tag v0.7.2
git push origin v0.7.2
```

3. Після push тегу, GitHub Actions автоматично запустить збірку для всіх платформ
4. Дочекайтеся завершення збірки та завантажте артефакти (як у Спосіб 1, крок 7)

### Спосіб 3: Через GitHub CLI (якщо встановлено)

```bash
# Встановіть GitHub CLI якщо не встановлено:
# brew install gh  (на macOS)
# або завантажте з https://cli.github.com

# Авторизуйтеся
gh auth login

# Запустіть workflow
gh workflow run "Build Desktop App.yml" -f platform=mac
```

## Як перевірити статус збірки

1. Перейдіть на вкладку **"Actions"** в репозиторії
2. Знайдіть ваш workflow run (найновіший в списку)
3. Натисніть на нього, щоб побачити деталі
4. Жовтий колір = в процесі, зелений = успішно, червоний = помилка

## Як завантажити зібраний додаток

1. Після успішної збірки, відкрийте workflow run
2. Прокрутіть вниз до секції **"Artifacts"**
3. Натисніть на назву артефакту (наприклад, `macos-dmg`)
4. Завантажте файл на ваш комп'ютер

## Примітки

- Збірка зазвичай займає 5-10 хвилин
- Артефакти зберігаються 30 днів
- Universal build для macOS працює на Intel та Apple Silicon
- Для Windows створюється один .exe файл
